# Changelog

## [1.0.0] - Initial Release

### Backend Features
- ✅ Full REST API with FastAPI
- ✅ PostgreSQL database with SQLAlchemy ORM
- ✅ JWT authentication and authorization
- ✅ Asset management (CRUD operations)
- ✅ Movement tracking
- ✅ Inventory sessions and results
- ✅ QR code label generation and PDF printing (20x30mm and 30x40mm)
- ✅ Excel and PDF report generation
- ✅ Reference data endpoints (companies, device types, warehouses, employees)
- ✅ Automatic inventory number generation (format: WWP-01/0030)
- ✅ Validation for unique serial numbers and inventory numbers
- ✅ Location validation (employee/warehouse)

### Frontend Features
- ✅ React + TypeScript + Vite setup
- ✅ Tailwind CSS for styling
- ✅ TanStack Query for data fetching
- ✅ JWT authentication with protected routes
- ✅ Assets page with search, list, and details
- ✅ Asset creation form
- ✅ Movements page with move functionality and history
- ✅ Inventory page with session management and scanning
- ✅ Print labels page
- ✅ Reports page with export functionality
- ✅ Bottom navigation (Kaspi-style)
- ✅ Header with user info and logout
- ✅ Responsive design (mobile-first)

### Infrastructure
- ✅ Docker Compose setup for development
- ✅ Dockerfiles for backend and frontend
- ✅ Environment configuration (.env files)
- ✅ Alembic database migrations
- ✅ CORS configuration

### Documentation
- ✅ README with setup instructions
- ✅ SETUP guide for detailed configuration
- ✅ API documentation via FastAPI /docs endpoint
